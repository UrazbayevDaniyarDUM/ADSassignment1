Description

This project contains solutions for 10 tasks using recursion in Java.
All tasks were implemented without using loops (for, while, do-while).

# Task 1 – Print Digits

Prints each digit of a number using recursion.

# Task 2 – Average of Elements

Calculates sum recursively and then finds average.

# Task 3 – Prime Number Check

Checks if a number is prime using recursion.

# Task 4 – Factorial

Calculates factorial of a number.

# Task 5 – Fibonacci

Finds n-th Fibonacci number.

# Task 6 – Power Function

Calculates a^n using recursion.

# Task 7 – Reverse Output

Reads numbers and prints them in reverse order.

# Task 8 – Check Digits in String

Checks if string contains only digits.

# Task 9 – Count Characters

Counts number of characters in string.

# Task 10 – GCD

Finds greatest common divisor using Euclidean algorithm.
